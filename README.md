            DLL Injector v2.0 - Nyxella & Wh1zz-52
========================================================================

ОПИСАНИЕ:
Инструментарий для динамической загрузки DLL в адресное пространство 
сторонних процессов. Проект разработан в образовательных целях для 
изучения механизмов Windows API и методов манипуляции памятью.

ОСОБЕННОСТИ:
Данный инжектор поддерживает 5 различных методов внедрения:

1. CreateRemoteThread (Classic) — стандартный метод.
2. NtCreateThreadEx (Undetected) — использование недокументированных 
   функций Native API для обхода простейших защит.
3. SetWindowsHookEx (Hook mode) — установка системного хука.
4. QueueUserAPC (APC injection) — асинхронный вызов процедур.
5. Thread Hijack (Advanced) — перехват существующего потока.

ФУНКЦИОНАЛ:
- Графический интерфейс на чистом Win32 API.
- Динамический список процессов с PID.
- Автоматический запрос прав администратора (UAC).
- Встроенное окно обзора файлов.

ТРЕБОВАНИЯ:
- ОС: Windows 7/8/10/11.
- Компилятор: MSVC (Visual Studio) или MinGW.
- Библиотеки: user32, comctl32, shell32, kernel32.

СБОРКА (MinGW):
g++ dll_injector.cpp -o injector.exe -luser32 -lcomctl32 -lshell32 -mwindows

Инструмент создан исключительно в ОБРАЗОВАТЕЛЬНЫХ целях. 
Используйте его только на собственных процессах.
